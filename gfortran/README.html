Welcome to the GNU Fortran 95 compiler!

This document intends to provide you with basic information about gfortran, the GUN Fortran 95 compiler, and links to further documentation about the compiler and the Fortran language.

Using gfortran

  • Our Getting Started page inside the gfortran wiki. This will give you the basics about gfortran (main command-line options, file extensions, etc.)
  • This distribution includes a copy of the gfortran documentation in HTML format and in PDF format (also available online, HTML or PDF). Other docs, such as the C compiler and preprocessor documentations, as well as the GCC and gfortran internals documentations, can be found in the doc/ directory.
  • This documentation includes a list of GCC general options (shared by gfortran), as well as options specific to gfortran

Getting help

There are a few places where you can find people willing to help you...
  • the comp.lang.fortran newsgroup on Usenet: people around there discuss fortran, and will be glad to offer you help and advice on any Fortran-related topic. Beware, though: it's know to be a place where discussions sometimes get heated!
  • the gfortran mailing-list (send a mail or search in the archives): it's the main place for discussions about gfortran. We'll help you use gfortran and answer your questions about the compiler.
  • the GCC bugzilla: if you found a bug in the compiler, please report it here with detailled instructions about how to reproduce it, the compiler version and your OS version.

Useful links

We maintain an online list of links and selected readings about Fortran. Please help yourself!

Source code for the utilities included in this package

This package contains more than just the GNU Fortran compiler. All the utilities included in here are open source, and as such, you are entitled to download their source code, look at it, modify it and redistribute it as long as you abide by their licence (which means, most of the times, that you cannot use them in such a way that doesn't offer the same freedom). For your convenience, and in keeping with the terms of the GNU Public Licence, here is the list of utilities included in this package and where you can download the source that was used to compiler them:

  • GNU binutils version 2.20 (source code here).
  • GNU gdb version 6.6 (source code here).
  • MinGW and MSYS runtimes environments, headers and development tools are public domain, and you can download their sources here.
  • pthreads-win32 version 2.8.0 (source code here).
  • Last but not least, this package is built with GCC development sources, which will become the 4.6.x release series. The revision number of the sources used for this build is 161045, and you can download those sources with Subversion as revision 161045 of svn://gcc.gnu.org/svn/gcc/trunk. This is usually done with the following commande line: svn -q checkout -r 161045 svn://gcc.gnu.org/svn/gcc/trunk gcc.
